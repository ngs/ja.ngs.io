# ja.ngs.io

長瀬敦史の日本語ブログ

https://ja.ngs.io/

## 技術スタック

- [Hugo](https://gohugo.io/) - 静的サイトジェネレーター
- [hugo-primer-blog](https://go.ngs.io/hugo-primer-blog) - テーマ (Hugo Module)
- GitHub Pages - ホスティング

## ローカル開発

```bash
# 開発サーバー起動
hugo server -D

# ビルド
hugo --gc --minify
```

## ライセンス

コンテンツの著作権は長瀬敦史に帰属します。
